Developer Operations Platform

Manage repos, scan health, clean artifacts, monitor ports, audit security, update dev tools.

Modules
Repository Scanner
Discover git repos, health-score them (A-F), detect containers, clean stale artifacts (node_modules, target/, .venv).
TUI
Repo Manager
ghq-compatible clone/pull/status. Organize by host/owner/name. Tag repos into groups. Pull by tag.
CLI TUI
Security Audit
4-phase scan: secrets, git history, OWASP code patterns, dependency vulnerabilities (OSV.dev + npm audit).
CLI TUI
Certificate Scanner
Scan SSL/TLS certs in Keychain and filesystem. Find expired certs, loose keys, with risk recommendations.
CLI
Port Scanner
Find and kill dev servers. Detect runtime (Node, Python, Go, Rust). Group dev vs system services.
TUI
System Cleanup
Docker, dev caches (brew/npm/pip/cargo), VMs, logs. Risk indicators. Safety guards inspired by tw93/mole.
TUI
Tool Updater
Manage 61 dev tools across 8 categories. Version comparison. Batch update with safety checks for runtimes.
TUI
Doctor
Validate installation health. Check binary, git, config, shell integration, completions, platform tools.
